作者：沐宇宇

     Django是Python一个重量级的web框架，与flask和tornado不同，django注重的是大而全，个人觉得里最突出的是它的admin后台管理，真的是太方便了。下面就来记录一下我学习django的过程。


1：安装django，在黑屏终端执行 ：
pip install django
2： 创建第一个项目：首先创建一个目录，cd进入该目录，执行：
django-admin stratproject 项目名
这样一个django项目就创建完毕了，目录结构应该如下图，对应文件的作用在后面会提到。

3：接下来就将我们的项目跑起来，进入manage.py文件所在目录，manage.py是每个Django项目中自动生成的一个用于管理项目的脚本文件， 执行：
python manage.py runserver
这样我们就启动了一个服务器，也就将我们的项目跑起来了。但是该服务器只是django提供的一个轻量级的服务器。
